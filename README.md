 Складской учёт
 Что умеет:
- Вести список товаров с автоматическим расчётом статуса (в наличии / мало / нет на складе)
- Управлять поставщиками
- Регистрировать складские операции (приход, расход, инвентаризация, возврат)
- Формировать отчёты и экспортировать их в CSV, HTML и PDF
- Отменять и повторять действия (Undo/Redo)
- Авторизация с разделением по ролям
- Поиск, фильтрация и цветовая индикация прямо в таблицах

Как запустить:

1. Открыть `Бачурин.sln` в Visual Studio
2. При первом открытии восстановить NuGet-пакеты (Newtonsoft.Json, PDFsharp)
3. Собрать решение (Ctrl+Shift+B) и запустить (F5)
   
Пароли для входа:

| Логин | Пароль | Роль | Что доступно |
|-------|--------|------|--------------|
| admin | admin | Администратор | Полный доступ |
| manager | manager | Менеджер | Всё, кроме удаления (для кладовщика) |
| worker | worker | Кладовщик | Товары и операции, без поставщиков |
Также есть кнопка **«Войти как гость»** — режим только для просмотра.

Структура проекта:

├── Models/        — модели данных (Product, Supplier, и др.)
├── Controllers/   — логика приложения
├── Services/      — хранение данных, экспорт, логирование
├── Forms/         — окна приложения
└── Program.cs     — точка входа
